# Retail Customer Behavior Analytics

## Project Overview
This project analyzes retail sales data to understand customer purchasing patterns, product performance, and regional sales trends. The goal is to derive insights that help businesses improve sales strategies and customer targeting.

## Business Problem
Retail companies need to understand which products sell the most, which regions generate the highest revenue, and which customers contribute the most to sales.

## Tools & Technologies
Python  
Pandas  
SQL  
Matplotlib  

## Dataset
The dataset contains retail transaction records including:
- Order ID
- Order Date
- Customer Name
- Product Category
- Region
- Sales
- Profit

## Project Workflow
1. Data Cleaning using Python
2. Exploratory Data Analysis
3. Business Queries using SQL
4. Data Visualization

## Key Analysis
- Total revenue generated
- Sales by product category
- Regional sales distribution
- Top 10 high-value customers

## Files
superstore.csv – Retail dataset  
analysis.py – Python data analysis script  
queries.sql – SQL queries for business analysis  

## Insights
- Technology products generated the highest revenue.
- Certain customers contribute significantly to total sales.
- Regional sales vary significantly across markets.
